Add a Filter to a Component

  1. In the Filters section of a component, click Add Filter > New.
  2. Enter a Name. It is helpful to make the name descriptive of what the filter matches. Names cannot be longer than 32 characters and can contain only letters, number, dashes, and underscores.
  3. Click Add Criteria, and then select a criterion to add to the filter. See
    • Data
    • Voice
    • Text
    for details about each type of filter criteria.
  4. Click Save.
If a voice or message filter includes a Remote Phone Number or if a message filter includes an SMS Body, Service Design Center checks the validity of the entered regular expression. If the regular expression is not valid, you will see an error and you will not be able to save the filter. See Regular Expression Tutorial and Examples for some basics on how to build valid regular expressions.
