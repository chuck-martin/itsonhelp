Edit an Entitlement Policy in a Network Feature

Editing an entitlement policy involves adding, editing, or deleting components or switching the entitlement code. 

You can edit an entitlement policy only when the network feature that contains it is in Draft or Test mode. Once a network feature that contains an entitlement policy has been published, you cannot edit the entitlement policy.

Add, Edit, or Delete a Component

Entitlement policy components contain only filters.
  • Click Component > New to add a component.
  • See the Change a Filter section in Edit a Policy in a Feature to edit the filter in an entitlement policy component.
  • Click  to delete a component.

Switch an Entitlement Code

  1. Next to Entitlement Code, click the name of the existing entitlement code, and then click Switch.
  2. Find the entitlement code you want to switch to, select it, and then click Add.
  3. Click the Unlock button  (to save and lock).

For editing an entitlement eCommerce policy, see Edit an eCommerce Policy.
