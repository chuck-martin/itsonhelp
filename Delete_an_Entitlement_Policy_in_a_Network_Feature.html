Delete an Entitlement Policy in a Network Feature

You can delete an entitlement policy only when the network feature that contains it is in Draft or Test mode. Once a network feature that contains an entitlement policy has been published, you cannot delete the entitlement policy.
  1. Click the Lock button  on the network feature page (to unlock and enable editing).
  2. Click the Delete button  in the entitlement policy you want to delete.
  3. Click Confirm.

For deleting an entitlement eCommerce policy, see Delete an eCommerce Policy.
