eMoney Allocation Customer Experience

Different combinations of the This product can be purchased for and the eMoney Allocation settings for eMoney create very specific sets of options on a client device at purchase time. This topic explains how you should configure these settings to determine what your customers can do with eMoney when they purchase it. Those options can be different for single-subscriber tenants and multi-subscriber tenants.

If you want to create an eMoney product that:

  • can be used only by the purchaser, select Customer's account only, and then Purchaser's subscriber wallet only.
  • can be used for the purchaser and any other subscribers in the account, select Customer's account only, and then Any subscriber wallet.
  • can be purchased only for account top-up, select Customer's account only, and then Account wallet only.
  • can be used for the purchaser, any other subscribers in the account, and for account top-up, select Customer's account only, and then Account wallet or any subscriber wallet.

If you want to also allow gifting for any of the above options, select Customer's account or gift recipient instead of Customer's account only.

If you want to create an eMoney product that is giftable only, select Gift recipient only (the eMoney Allocation setting does not matter).
