Edit a Network Group

Network groups are used in policies "by reference." A change to a network group definition changes that definition in every policy that uses (references) the network group. 

If the network group is being used by any published policies, you will be asked if you want to update devices in the field immediately. Because network groups are shared across policies, make sure you know what policies any network group changes will affect.

  1. Go to Libraries > Network Groups.

  2. Find the network group in the list that you want to edit, and then click the corresponding View button  in the right column. (The buttons in the right column appear when you hover the mouse pointer over the row.)

  3. Click the Unlock  button (to unlock and enable editing).
  4. Make any desired changes. See Network Group Settings for details.
    When you press the space bar, Tab key, or Enter key, or click into a different field after entering network group values, the entries are validated automatically.
  5. Click Save.

The Save button is not active if you have not entered any SIDs, NIDs, MCCs, or MNCs or if you have invalid ones entered and they are not corrected or cleared.
