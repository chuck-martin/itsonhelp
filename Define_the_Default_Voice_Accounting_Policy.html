Define the Default Voice Accounting Policy

When you add a voice service policy to a feature, it gets the default voice accounting policy associated with it. (If there is only one voice accounting policy in the library, that one is automatically the default.) If you have more than one voice accounting policy in the library, you can define which one will be used as the default in new voice service policies.

  1. Go to Policy > Advanced Voice Accounting Policy.
  2. Click next to the voice accounting policy you want to make default.
  3. In the confirmation dialog box, click Confirm.
The selected voice accounting policy will be the default for all newly created voice service policies. This does not change the associations of existing voice service policies, including any voice service policies that referenced the previous default voice accounting policy.

You must refresh the page to see the change.
