Network Features vs. eCommerce Policies

In Service Design Center,  you can create 2 types of entitles that contain components, filters, policy events, voice accounting policies, bucket IDs, and rating groups: network features and eCommerce policies. This topic explains the differences.

Entity Unique Characteristics
Network Feature
  • Meant for network management.
  • Can contain one or more of any type of policies.
  • Once published, appear on the Provisioning page.
  • When publishing, can choose when ItsOn-powered devices receive the feature.
eCommerce Policy
  • Meant to contain policy configuration that policy-based products, such as plans and entitlement products, are based on.
  • Is either a single service policy or a single entitlement policy.
  • Once published, the policies are available to be added to products.
