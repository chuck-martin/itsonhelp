View a Category

When you view a category, you view it in the context of a sandbox. This means that you are viewing the category's settings as it exists as deployed, plus any undeployed changes made to the category in that sandbox.

  1. Go to eCommerce > Catalog & Notifications.
  2. Select the Sandbox view you want to work in. Hover over the down arrow next to the current sandbox name in the header, then click a sandbox name.
  3. Go to Catalog > Categories.
  4. Click the name of a category.

If a category has changes in it in that sandbox, the category name will be yellow and there will be a pencil icon next to the category name, like this:

On a category's detail page, any fields that have been changed from the deployed version will be outlined in the color of the sandbox.
